# React Redux
- It is a state management library
- It has only 5 major functions: 
1. create store
2. combine reducer
3. compose
4. apply middleware
5. bind action creators

## Installing React-Redux
Use the following commands to install redux
```
npm init
```
```
npm install redux
```
1. Create `index.js` file
